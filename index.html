Preencha os dados abaixo para validar se o produto pode ser cadastrado no marketplace.

Regras de validação do MVP

  • ✅ O preço deve ser maior que zero.
  • ✅ O título do produto deve ter pelo menos 5 caracteres.
  • ✅ A categoria deve ser uma das permitidas: Fruta, Legume ou Verdura.

Essas regras fazem parte da lógica de negócio definida pelo time da GreenEats para evitar erros de cadastro no marketplace.

Exemplo de payload enviado ao backend:

{
  "titulo": "Maçã orgânica Fuji",
  "preco": 5.90,
  "categoria": "Fruta"
}

Este protótipo usa HTML, CSS e JavaScript no frontend e uma API em Python (Flask) integrada ao Airtable no backend, conforme solicitado na avaliação.
